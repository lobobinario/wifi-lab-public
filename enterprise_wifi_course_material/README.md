# Material docente — Enterprise WiFi Security Lab

Recursos para impartir la sesión de seguridad WiFi tipo empresa.

## Contenido

- **`guion_instructor.md`** — Guion paso a paso para el instructor con los comandos exactos y los mensajes clave de cada bloque.
- **Slides** — Se encuentran en `../slides/index.html`. Abrir directamente en el navegador o servir con `python -m http.server` desde el directorio `slides/`.

## Uso

1. Repasar `guion_instructor.md` antes de la clase.
2. Abrir las slides en pantalla completa (atajo `F` activa el modo focus).
3. Seguir el guion alternando entre las slides y la terminal de la Raspberry Pi.
